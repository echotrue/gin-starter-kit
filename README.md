# gin-starter-kit
Gin framework starter kit

This project need `go mod` support

#### Getting Started

- Clone code
```shell script
git clone ...
```

- Install dependency package
```shell script
go get 
```

- Run
```shell script
fresh
```

#### Dependency
The kit uses the following Go packages    
- log : `logrus`
- db : `ozzo-dbx`
- jwt :`gin-jwt`
- redis: `redigo`
- config : `toml`
